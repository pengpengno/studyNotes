linux 命令行
查找文件 模糊查询
find / -name redis*
查找redis-server
find / -name redis-server

创建 文件夹 java
mkdir  java

linux下重命名文件或文件夹的命令mv既可以重命名，又可以移动文件或文件夹.
例子：将目录A重命名为B
mv A B
例子：将/a目录移动到/b下，并重命名为c
mv /a /b/c
